# YAL (Yet Another Language)

A Turing-complete interpreted language written in Go.

## Features
- Modular architecture
- MIT License
- Comprehensive tests
- GitHub Actions CI for build and test

## Getting Started

```sh
git clone https://github.com/yourusername/yal.git
cd yal
go build ./...
```

## Running Tests

```sh
go test ./...
```
